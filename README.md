
# Sistema de Aluguel de Carros — Scripts SQL

Repositório contendo todos os scripts para criação e testes do banco de dados.

## Como executar no DB-Fiddle

1. Entre em https://db-fiddle.com
2. Escolha MySQL 8.0
3. Cole os arquivos na seguinte ordem:
   - create_tables.sql
   - inserts.sql
   - select_examples.sql
   - delete_examples.sql

## Estrutura das Tabelas
- MOTORISTA
- VEICULO
- CONTRATO
- CHECKLIST
